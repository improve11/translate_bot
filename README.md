# 🤖 Telegram Translator Bot

[![Node.js](https://img.shields.io/badge/Node.js-18%2B-green.svg)](https://nodejs.org/)
[![Telegraf](https://img.shields.io/badge/Telegraf-4.x-blue.svg)](https://telegraf.js.org/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

**Многофункциональный бот-переводчик для Telegram** с поддержкой автоопределения языка, ручной настройки исходного и целевого языков, а также хранением пользовательских настроек в JSON-файле. Работает на Node.js с использованием библиотеки `telegraf` и неофициального API Google Translate.

---

## ✨ Возможности

- 📥 **Перевод текста** – отправьте любое сообщение, бот переведёт его на заданный язык.
- 🔍 **Автоопределение исходного языка** – бот сам определяет, с какого языка переводить.
- 🎯 **Настройка целевого языка** – выберите язык, на который хотите переводить (например, русский, английский, испанский и др.).
- 📝 **Ручная установка исходного языка** – если автоопределение не нужно, можно указать язык вручную.
- 🗂️ **Хранение настроек** – ваши предпочтения сохраняются в JSON-файле и не теряются при перезапуске бота.
- 📋 **Просмотр текущих настроек** – команда `/mylang` показывает все ваши настройки.
- 📚 **Список поддерживаемых языков** – команда `/languages` выводит все доступные языки с кодами.

---

## 🛠️ Технологии

- [Node.js](https://nodejs.org/) – среда выполнения JavaScript.
- [Telegraf](https://telegraf.js.org/) – современный фреймворк для Telegram Bot API.
- [@vitalets/google-translate-api](https://github.com/vitalets/google-translate-api) – бесплатный (неофициальный) клиент Google Translate (версия **7.0.0**, совместимая с CommonJS).
- [dotenv](https://github.com/motdotla/dotenv) – для управления переменными окружения (токен бота).
- Встроенный модуль `fs` – для работы с JSON-хранилищем пользователей.

## 📦 Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/telegram-translator-bot.git
cd telegram-translator-bot
```
### 2. Установка зависимостей
Установите все необходимые пакеты (обратите внимание на фиксированную версию @vitalets/google-translate-api, чтобы избежать ошибок):

```bash
npm install
# или вручную:
npm install telegraf dotenv @vitalets/google-translate-api@7.0.0
```
## 📖 Команды бота

Команда	Описание
/start	Приветственное сообщение и список основных команд.
/help	Подробная справка по использованию.
/setlang <код>	Установить целевой язык перевода (например, /setlang ru).
/setfrom <код>	Установить исходный язык (например, /setfrom en). Если указать auto, включится автоопределение.
/autodetect	Включить автоопределение исходного языка (режим по умолчанию).
/mylang	Показать текущие настройки (целевой язык, исходный язык, статус автоопределения).
/languages	Вывести список всех поддерживаемых языков с кодами.
Пример:
Чтобы переводить все сообщения на русский язык, отправьте:

```bash
/setlang ru
```

Теперь любой текст, который вы пришлёте боту, будет переведён на русский.# translate_bot
